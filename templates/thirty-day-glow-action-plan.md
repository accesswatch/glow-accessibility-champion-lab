# 30-Day GLOW Action Plan Template

Use this template for a focused 30-day sprint toward one GLOW workflow deliverable.

## Goal

What specific deliverable will be complete in 30 days?

## Week 1: Ground and Learn

- [ ] Identify the real accessibility problem.
- [ ] Define the campus partner audience.
- [ ] Identify the accessibility pattern behind the problem.
- [ ] Select a low-risk task with public or sanitized content.

## Week 2: Organize

- [ ] Draft the GLOW workflow using the template.
- [ ] Write the structured prompt.
- [ ] Run the prompt and save the raw output.
- [ ] Review and edit the output.

## Week 3: Refine

- [ ] Run the fairness review checklist.
- [ ] Run the human review checklist.
- [ ] Revise the prompt if needed.
- [ ] Test with a realistic example.

## Week 4: Walk Forward

- [ ] Finalize the reusable artifact.
- [ ] Write the teaching note or facilitator guide.
- [ ] Commit and document the workflow.
- [ ] Plan the next step or pilot.
